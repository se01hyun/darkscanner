# 🛡️ Dark-Scanner PRD (Product Requirements Document)

## 1. 프로젝트 개요
* **프로젝트 명**: Dark-Scanner (다크 스캐너)
* [cite_start]**목표**: 온라인 쇼핑몰에서 소비자의 합리적 선택을 방해하는 **'오도형'** 및 **'압박형'** 다크패턴을 실시간 탐지하는 크롬 확장 프로그램[cite: 170].
* [cite_start]**핵심 가치**: 소비자가 자신의 선호에 따라 자유롭고 합리적으로 의사결정을 할 수 있는 인터페이스 환경 조성[cite: 175].

---

## 2. 탐지 대상 및 범위 (Scope)
[cite_start]정부 가이드라인의 4개 범주 중, 텍스트 분석과 시각적 요소 판별이 용이한 두 가지 범주에 집중합니다[cite: 74, 172].

### 🟢 오도형 (Misleading)
[cite_start]거짓을 알리거나 통상적인 기대와 다르게 화면을 구성해 착각을 유도하는 유형[cite: 173].
* [cite_start]**거짓 할인**: 할인 정보를 거짓으로 표시해 높은 가격에 구매하도록 유도 (예: 부풀린 기준 가격)[cite: 216, 222].
* [cite_start]**특정옵션 사전선택**: 사업자에게 유리한 옵션을 미리 선택해두고 소비자가 무심코 수용하게 함[cite: 272].
* [cite_start]**잘못된 계층구조**: 유리한 선택지만 시각적으로 두드러지게 표시 (예: 취소 버튼을 회색으로 숨김)[cite: 263, 268].

### 🔴 압박형 (Pressure)
[cite_start]심리적 압박을 가해 특정 행위를 하거나 하지 않도록 유도하는 유형[cite: 173].
* [cite_start]**감정적 언어사용**: "혜택 포기하기" 등 감정을 자극하는 표현으로 특정 행동 압박[cite: 337, 338].
* [cite_start]**시간제한 알림**: 거짓된 제한 시간을 표시하여 의사결정을 재촉함[cite: 342, 343].
* [cite_start]**다른 소비자의 활동 알림**: 최근 구매자 수 등을 표시하여 구매 심리 압박 (실태조사 결과 사용 빈도 1위 유형)[cite: 11, 354].

## 3. 주요 기능 요구사항

### 3.1 실시간 페이지 스캔 (Content Script)
* [cite_start]쇼핑몰 웹사이트 접속 시 DOM 구조를 분석하여 버튼, 가격, 체크박스 상태 데이터를 추출함[cite: 6].
* [cite_start]동적으로 변하는 카운트다운 타이머나 팝업창을 실시간 감지함[cite: 322].

### 3.2 AI 기반 패턴 분석
* [cite_start]**텍스트 분류 (NLP)**: 추출된 문구가 '감정적 언어'인지, '압박성 문구'인지 Claude API 등을 활용해 분류함[cite: 337].
* [cite_start]**시각적 요소 검증**: 버튼의 색상 대비(Contrast)나 크기 차이를 분석하여 '잘못된 계층구조' 여부 판별[cite: 267, 379].

### 3.3 사용자 경고 인터페이스 (UI)
* [cite_start]**요소 하이라이트**: 다크패턴으로 의심되는 UI 요소에 경고 표식(레드 보더 등)을 생성함[cite: 113].
* [cite_start]**정보 툴팁**: 해당 패턴이 가이드라인의 어떤 유형에 해당하며, 왜 주의해야 하는지 근거를 제시함[cite: 176].

## 4. 기술 스택 (Proposed)
* **Browser**: Chrome Extension Manifest V3
* **Language**: JavaScript
* **Analysis**: Claude API (패턴 분류 및 분석 로직)
* [cite_start]**Reference**: 공정거래위원회 '온라인 다크패턴 자율관리 가이드라인'[cite: 169].

## 5. 성공 지표
* [cite_start]**정확도**: 실태조사에서 언급된 주요 다크패턴 사례(평균 5.6개 유형 사용)에 대해 높은 탐지율 달성[cite: 7].
* [cite_start]**사용성**: 사용자가 쇼핑 과정을 방해받지 않도록 신속하고 직관적인 경고 제공[cite: 174].
