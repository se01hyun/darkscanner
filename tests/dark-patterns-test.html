Dark-Scanner 탐지 테스트 페이지

크롬에서 Dark-Scanner 확장 프로그램을 활성화한 뒤 이 페이지를 열면
아래 항목에 빨간 테두리 여부로 탐지 정확도를 확인할 수 있습니다.

P1-A: 다른 소비자의 활동 알림

탐지되어야 함
  • 153명이 보는 중
  • 최근 24명이 구매했습니다
  • 지금까지 3,200개 구매
  • 47명이 장바구니에 담았습니다
  • 재고 3개 남음
  • 오늘 82건 주문
  • 한정 10석
  • 품절 임박
  • 지금 막 구매
탐지되면 안 됨 (오탐 검증)
  • 내일(화) 4/7 도착 보장 (7시간 39분 내 주문 시 / 서울·경기 기준)
  • 배송비 무료
  • 카드 할인 적용 가능
  • 상품 설명을 확인하세요
  • 11 개 상품평
  • 로그인 후 이용해주세요
  • 지금 보고 계신 화면은 결제 완료 페이지입니다

* "39분 내 주문"이 탐지되면 SOCIAL_PROOF_PATTERN 오탐
* "지금 보고 계신 화면은..."이 탐지되면 지금 보고 단독 패턴 오탐 — 숫자 없는 일반 문장이므로 탐지 불가여야 함

P1-B: 감정적 언어사용

탐지되어야 함 (규칙 기반 — API 키 불필요)

카테고리 1 — 혜택·기회 포기 프레이밍

카테고리 2 — 자기비하적 거절

카테고리 3 — 무시·경시 프레이밍

카테고리 4 — 직접적 손실 인정

탐지되면 안 됨

* "할인 받기 싫어요"는 규칙 33개 중 매칭되는 패턴 없음 — 탐지되면 오탐

P1-C: 시간제한 알림

탐지되어야 함
  • 오늘만 할인
  • 마감 임박
  • 이 가격 마지막
  • 3시간 남음
  • 20분 이내
  • D-2
  • 3일 후 종료
  • 오늘 오후 6시 마감
  • 타임세일
  • flash sale
  • 타이머: 02:59
탐지되면 안 됨
  • 내일 도착 보장
  • 평일 오후 2시 이전 주문 당일 출고
  • 무료배송
  • 평균 응답 시간은 약 3시간 남음이 있습니다
  • 재고가 3개 남았습니다만 추가 입고 예정입니다

* "남음이", "남았습니다" 처럼 조사·어미가 붙으면 서술어이므로 탐지되면 안 됨
(조사 없이 끝나는 카운트다운 표현만 탐지 대상)
